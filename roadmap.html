Roadmap

V0.12.0 is out. The next work is making installation, AI setup, and daily use easier to trust.

Now This cycle

  1. Polish first-run guidance: install path, kaku doctor, shell integration, and optional tool setup.
  2. Keep the AI chat loop tight: context window, model routing, session titles, and web-fetch summaries.
  3. Stabilize AI provider, proxy, and reasoning-stream handling across providers.

Next Next cycle

  1. Add more provider presets and clearer troubleshooting guidance.
  2. Continue hardening the AI tool sandbox and file-access policy.
  3. Improve macOS distribution, update, and first-launch experience.

Later Idea pool

  1. Evaluate non-macOS platforms without promising a timeline.
  2. Deeper session restore, recording, and replay.
  3. More IDE, tmux, and remote-development integrations.

Release notes live in GitHub Releases. Implementation details still live in GitHub Issues.
