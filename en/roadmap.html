Roadmap

Kaku is becoming a quieter AI terminal.

The roadmap stays short and honest. Implementation details still live in GitHub Issues.

Now

This cycle

  • Ship the website v1 and make `kaku.tw93.fun` reliable.
  • Polish error recovery and natural-language command feedback.
  • Stabilize AI provider, proxy, and fast-model paths.

Next

Next cycle

  • Expand migration docs and configuration examples.
  • Improve macOS distribution, update, and first-launch experience.
  • Add more provider presets and troubleshooting guidance.

Later

Idea pool

  • Evaluate non-macOS platforms without promising a timeline.
  • Deeper session restore, recording, and replay.
  • More IDE, tmux, and remote-development integrations.
