Error recovery

A failed command should be more than red text.

Kaku sends the failed command, exit code, working directory, and git branch to Assistant, then returns a reviewable fix. Press Cmd + Shift + E to paste it back; dangerous commands are never auto-run.

Natural language to command

Let the terminal understand a sentence without running it behind your back.

Type # plus a plain-language request at the prompt. Kaku calls your configured LLM and injects the generated command back for review.

Entrypoints you actually use every day.

The site stays close to the product docs and code instead of inventing a separate marketing story.

01

Chat

Cmd + L opens AI Chat; use k or kaku chat from the shell to continue the same conversations and memory.

02

Tools

Cmd + Shift + G opens Lazygit, Cmd + Shift + Y opens Yazi, and Cmd + Shift + R mounts remote files inside SSH sessions.

03

Doctor

kaku doctor checks the app bundle, PATH, shell integration, and optional tools; kaku init provisions zsh/fish integration and common tools.
