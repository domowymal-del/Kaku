Error recovery

A failed command should be more than red text.

Kaku suggests a reviewable fix after command failures. You stay in control: the suggestion only returns to the prompt when you apply it.

Natural language to command

Let the terminal understand a sentence without running it behind your back.

Type # plus a plain-language request at the prompt. Kaku calls your configured LLM and injects the generated command back for review.
