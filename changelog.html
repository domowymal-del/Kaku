Changelog

Each release should say what changed. GitHub Releases stay the source of truth for full history and assets.

V0.12.0 Sharper Upcoming

  1. AI prompts: rebuilt into versioned fragments with a smarter chat loop, so /suggest, session titles, and the web-fetch summarizer all route to the fast model.
  2. AI shell hardening: the # quick-fix gains command-injection detection, and the file-access sandbox is audited for symlink and credential-path escapes.
  3. macOS: Light/Dark flips refresh every window at once, documents open in their default app, and Cmd+Q smart-confirms, quitting at a bare shell prompt and asking when an agent or editor is running.

V0.11.0 Steady 2026-05-17

  1. Reasoning streams: DeepSeek-compatible, GLM, Kimi, and Fireworks streams stay hidden so final answers stay clean, and kaku chat shows a compact thinking state.
  2. Safer init: kaku init handles read-only config paths, existing jump providers, and Yazi themes, and new session-restore settings make window snapshots easier to control.
  3. Window and mouse polish: Cmd+W closes the last fullscreen tab correctly, title-bar dragging is steadier, and top-tab hit testing is more stable.

V0.10.0 Chat 2026-05-10

  1. AI Chat, one engine and two surfaces: Cmd + L opens a panel inside Kaku, and k / kaku chat run the same engine as a TUI anywhere, with streaming Markdown, project tools, web search, and memory.
  2. Smart close protection: Cmd + W / Cmd + Shift + W ask before killing a pane running an agent or editor, and bare shells still close silently.
  3. Window snapshots: multi-tab and multi-pane layouts auto-save with Cmd + Option + Shift + T, plus a full AppleScript dictionary and animated tab drag.

V0.9.0 Spark 2026-04-04

  1. Natural language to command: type # plus a request and Kaku injects the generated command back into the shell.
  2. Cursor jumps: Option + Click moves the cursor within the current command line, including wide and multi-byte characters.
  3. New options: always-on-top windows, configurable traffic light position, and a MiniMax provider preset.
