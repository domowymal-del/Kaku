更新日志

每个版本都应该说明它改变了什么。完整版本历史和下载资产以 GitHub Releases 为准。

V0.12.0 Sharper 即将发布

  1. AI 提示：重构为带版本号的片段，对话回路更利，/suggest、会话标题、网页摘要都走 fast model。
  2. AI Shell 加固：# 快速修复加入命令注入检测，文件访问沙箱补充软链逃逸与凭证路径测试。
  3. macOS：浅/深色切换一次性刷新所有窗口，文档用系统默认 app 打开，Cmd+Q 智能确认，所有面板在 shell 提示符时直接退出，仍有 agent 或编辑器运行时先询问。

V0.11.0 Steady 2026-05-17

  1. 推理流：DeepSeek 兼容、GLM、Kimi、Fireworks 的推理流保持隐藏，最终回答更干净，kaku chat 显示紧凑 thinking 状态。
  2. kaku init 更安全：对只读配置路径、已有目录跳转工具和 Yazi 主题更稳，新增会话恢复设置让窗口快照更易控制。
  3. 窗口与鼠标细节：Cmd+W 正确关闭最后一个全屏标签，标题栏拖动更稳，顶部标签命中测试更可靠。

V0.10.0 Chat 2026-05-10

  1. AI 对话，一套引擎两个入口：Cmd + L 在 Kaku 内打开面板，k / kaku chat 把同一套引擎放进任意终端的 TUI，流式 Markdown、项目工具、网页搜索、本地记忆齐全。
  2. 智能关闭保护：Cmd + W / Cmd + Shift + W 在关闭运行 agent 或编辑器的 pane 前先询问，纯 shell 仍静默关闭。
  3. 窗口快照：自动保存多标签多面板布局（Cmd + Option + Shift + T），并提供完整 AppleScript 字典和标签拖动动画。

V0.9.0 Spark 2026-04-04

  1. 自然语言转命令：在提示符前输入 # 描述，Kaku 生成命令并回填到 shell。
  2. 光标跳转：Option + Click 支持在当前命令行移动光标，正确处理宽字符和多字节输入。
  3. 新增选项：窗口置顶、Traffic Lights 位置配置和 MiniMax Provider 预设。

完整 release notes 与 DMG 下载都在 GitHub Releases。
