路线图

V0.12.0 已经发布。接下来重点是让安装、AI 设置和日常使用更清楚、更可靠。

Now 本轮正在做

  1. 打磨首次上手路径：安装方式、kaku doctor、shell 集成和可选工具准备。
  2. 保持 AI 对话回路紧凑：上下文窗口、模型路由、会话标题和网页摘要。
  3. 稳定各 Provider 的配置、代理和推理流处理。

Next 下轮规划

  1. 增加更多常见 Provider 预设，并补齐更清楚的排查提示。
  2. 继续加固 AI 工具沙箱和文件访问策略。
  3. 优化 macOS 分发、更新和首次启动体验。

Later 想法池

  1. 评估非 macOS 平台，不承诺时间表。
  2. 更完整的会话恢复、录制和回放能力。
  3. 围绕 IDE、tmux 和远程开发做更深集成。

发布说明见 GitHub Releases。具体问题和实现讨论仍然以 GitHub Issues 为准。
